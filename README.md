# 📧 Email AI Assistant

An AI-powered **Google Chrome extension** that helps users **automate email replies** and **rewrite drafts** into clear, professional messages.
Built with **Spring Boot** and **Spring AI**, this project shows how I can combine backend development with a real-world browser extension to solve everyday communication problems.

---

## 💡 Why I Built This

Writing professional emails can be time-consuming and stressful. I created this assistant to:

* Save time by generating **automatic replies**
* Help users **rewrite emails** to improve clarity and tone

---

## ⚙️ How It Works

1. User drafts or pastes an email in the Chrome extension.
2. Select **“Rewrite”** → AI improves tone, clarity, and professionalism.
3. Select **“Reply”** → AI generates a suggested response.
4. Copy & paste directly into Gmail, Outlook, or any email client.

## 🏗️ Quick Start

1. **Clone the repository**

   ```bash
   git clone https://github.com/Theinhan10/Email-AI-Assistant.git
   cd Email-AI-Assistant
   ```

2. **Set up the backend (Spring Boot)**

   * Navigate to the backend folder:

     ```bash
     cd EmailAIAssistant
     ```
   * Add your API key to a `.env` file:

     ```
     OPENAI_API_KEY=your_api_key_here
     ```
   * Run the backend:

     ```bash
     mvn spring-boot:run
     ```

3. **Load the Chrome Extension**

   * Open Chrome → Go to **Extensions** → Enable **Developer mode** → Click **Load unpacked**
   * Select the `Email-Assistant-Extension/` folder
